# Act_4

# Pilas (Stacks) – Verificador de Sintaxis Avanzado


## Descripcion

Desarrollar un programa en Java que procese cadenas de texto multimedia ( expresiones matemáticas con paréntesis, corchetes y llaves, o fragmentos de código HTML). El sistema determinara si todos os símbolos de apertura y cierre están correctamente balanceados y 
respetan la jerarquía de anudamiento. El programa indicara la posición exacta (linea y 
columna) de cualquier error detectado, como cierres sin apertura, aperturas sin cierre o discrepancias de jerarquía.
